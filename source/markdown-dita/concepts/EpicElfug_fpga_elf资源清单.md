# FPGA-Elf资源清单 {#A5354535-51F2158-7855171CA927BD7-3D0386 .concept}

-   逻辑资源：Kintex KU040；
-   逻辑单元（Logic Cells）：530K；
-   查找表（\(LookUp Table，LUTs）：242,400；
-   触发器（Flip-flops）：484,800；
-   乘法器：1920；
-   Block RAM：21.1Mb；
-   20路高速GTX收发器，支持PCIe Gen3 8-lane。

详细数据见Xilinx DataSheet。
